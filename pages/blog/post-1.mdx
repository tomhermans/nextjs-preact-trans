## Theme UI

[Theme UI](https://theme-ui.com/home/) gives you a base set of [UI components](https://theme-ui.com/components) that let me style and layout web components with ease. The [sx prop](https://theme-ui.com/sx-prop) is magic. No thinking about class names, the cascade or how my CSS is going to scale. Write your styles, then get minified, atomic classes and automatically generated [critical CSS](https://web.dev/extract-critical-css/) for your pages. Great for devs, and great for users.
